### 📡 Network Traffic Analyzer
> Python · Scapy · Wireshark

A real-time packet capture and analysis tool for detecting active network attacks.

**What it does:**
- Detects ARP spoofing, port scanning, and DNS tunnelling in real time
- Decodes TCP/IP, HTTP, and DNS protocol layers to surface unencrypted credentials and anomalous traffic

**Validated in:** Isolated VirtualBox home lab (Kali Linux + Metasploitable 2) — successfully flagged Nmap SYN scans and live arpspoof attacks
